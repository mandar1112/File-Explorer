# File Explorer

A modern File Explorer built with Python and PySide6.


## Features

### Navigation
- Browse directories
- Sidebar navigation
- Address bar
- Back navigation
- Forward navigation
- Up navigation

### File Operations
- Open files and folders
- Create new folder
- Create new file
- Rename files and folders
- Delete files and folders
- Copy
- Cut
- Paste

### User Interface
- Menu Bar
- Toolbar
- Status Bar
- Sidebar
- Context Menu
- Background Context Menu
- Properties Dialog
- About Dialog

### Architecture
- Modular MVC-inspired architecture
- Controllers
- Services
- Reusable Dialogs
- Utility modules

---



## Technologies

- Python
- PySide6
- pathlib
- shutil
- os
- sys
