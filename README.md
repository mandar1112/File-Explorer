# File Explorer

A modern File Explorer built with Python and PySide6.


## Features

- Browse directories
- Sidebar navigation
- Back / Forward / Up navigation
- Open files
- Rename files and folders
- Delete files and folders
- Menu Bar
- Toolbar
- Context Menu


## Technologies

- Python
- PySide6
- pathlib

## Status

Under active development.
