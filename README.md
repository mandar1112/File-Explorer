# File Explorer

A modern File Explorer built with *Python* and *PySide6*, featuring a modular MVC-inspired architecture.


## Features

### Navigation
- Browse directories
- Sidebar navigation
- Address bar
- Back navigation
- Forward navigation
- Up navigation

### File Operations
- Open files and folders
- Create folder
- Create file
- Rename
- Delete
- Copy
- Cut
- Paste

### User Interface
- Menu Bar
- Toolbar
- Status Bar
- Sidebar
- Context Menu
- Background Context Menu
- Properties Dialog
- About Dialog

### Architecture
- MVC-inspired architecture
- Controllers
- Services
- Reusable Dialogs
- Utility modules

---



## Technologies

- Python
- PySide6
- pathlib
- shutil
- os
- sys
